## ✨ Features

🚀 **Next-Gen Development Experience**
- AI-powered code assistance with Claude integration
- Real-time collaboration and agent-driven development
- Cross-platform desktop app (Linux, macOS, Windows)
- Mobile IDE companion for iOS

🎨 **Modern Interface**
- Canvas-based project visualization
- Terminal integration with custom commands
- File tree with intelligent navigation
- Unified task and backlog management

🔧 **Developer Tools**
- Git integration with diff management
- Background agent execution
- Memory tracking and performance optimization
- Extensible through custom scripting

> 💡 **nvm users**: Set Node.js 24 as default: `nvm alias default 24`

### 🏃‍♂️ One-Line Setup

```bash
# Install Just task runner
npm install -g just
```
